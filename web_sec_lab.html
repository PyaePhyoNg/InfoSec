Web Application Security Guide for Beginner

Before You Start

  • Information Security field ထဲကို အမှန်အကန်စဝင်တော့မယ်ဆိုရင် Hacking နဲ့ ပက်သက်ပြီး တစ်ချိန်ကသိခဲ့သမျှကို မေ့ထားဖို့တော့လိုလိမ့်မယ်။
  • Penteration Testing ဆိုတဲ့အကြောင်းကို မဖတ်ဖူးသေးပဲ မသိနားမလည်သေးခင် Hacking Tool တွေကို မသုံးပါနဲ့။
  • Tool မှာရေးထားတဲ့ code ကို နားမလည်ပဲ မဆန်းစစ်ပဲ download လုပ်ပီး မသုံးမိစေနဲ့။
  • စာဖတ်ပါ၊ အသုံးပြုပုံတွေလေ့လာပါ၊ Tool တွေကို ဘုမသိဘမသိ လျှောက်သုံးနေတဲ့ Scritpt kiddie မဖြစ်ပါစေနဲ့။
  • Hacker အမျိုးအစားတွေကို ဒီမှာ သွားဖတ်ကြည့်ပါ။
  • Hacking ပညာရပ်တွေထဲက web application hacking ကို စပြီးလေ့လာဖို့အတွက် Reference အနေနဲ့ Web Application Hacker's Handbook (2nd Edition)ကို အဓိကထားပြီး သတိရတိုင်းဖတ်ပါ ပြီးဆုံးအောင်ဖတ်ပါ နားလည်အောင်ဖတ်ပါ။ အခြေခံသဘောတရားနည်းလမ်းတွေသိလာြပီ ဆိုမှ နောက်ထပ်ဖတ်ရမဲ့ စာအုပ်တွေကို ထပ်ညွှန်းပေးပါမယ်
  • စာအုပ်ဖတ်ရင်းနဲ့ ပဲ အောက်မှာပေးထားတဲ့ lab တွေနဲ့ တွဲပြီးလေ့ကျင့်ပါ။
  • Lab သွင်းတဲ့အချိန် အဆင်မပြေတာကြုံတွေ့လာရင် Internet ပေါ်မှာ အရင်ရှာကြည့်ပြီး ဖြေရှင်းပါ။ အဲ့လိုဖြေရှင်းလို့ မရတော့ဘူး မလုပ်တတ်တော့ဘူးဆိုမှလာမေးပါ။ ထပ်ပြီးလမ်းညွှန်ပေးပါမယ်
